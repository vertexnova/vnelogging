# GitHub Wiki Content

These markdown files are templates for the [vnelogging GitHub Wiki](https://github.com/vertexnova/vnelogging/wiki).

## How to use

### Option 1: Clone the wiki repo and copy

1. Enable the Wiki in your repo: **Settings → General → Features → Wiki** (check it)
2. Clone the wiki: `git clone https://github.com/vertexnova/vnelogging.wiki.git`
3. Copy the `.md` files from this `docs/wiki/` folder into the cloned wiki repo
4. Commit and push: `git add . && git commit -m "Add wiki pages" && git push`

### Option 2: Create pages manually

1. Go to the [Wiki](https://github.com/vertexnova/vnelogging/wiki) tab
2. Create each page (e.g. "Quick-Start", "Installation")
3. Copy-paste content from the corresponding `.md` file here
